# Build-up Questions (One-Line Answers)

## 1. Does higher GHz always mean a faster CPU?
No, IPC and CPU architecture also affect performance.

## 2. Why do modern CPUs have multiple cores?
To execute multiple tasks simultaneously.

## 3. What is the difference between a core and a thread?
A core is physical hardware; a thread is a logical execution path.

## 4. Why is cache much faster than RAM?
Because it is built inside the CPU and located much closer to the execution units.

## 5. What is latency?
The time taken to complete a single operation.

## 6. What is throughput?
The amount of work completed per unit time.

## 7. Which is better: low latency or high throughput?
Both are important—latency improves response time, while throughput improves overall capacity.

## 8. What is a system bottleneck?
The slowest component that limits overall performance.

## 9. How can you check CPU information in Linux?
`lscpu`

## 10. Which command shows CPU and memory usage in real time?
`top` or `htop`
